# Security Policy

## Reporting a vulnerability

Please report security issues via GitHub Security Advisories.

- Open a private security advisory in this repository.
- Provide a clear description, impact, and reproduction steps.

## Scope

We prioritize reports involving:

- approval/consent bypass
- policy or capability escalation
- sandbox or tool boundary escape
- data exfiltration or unintended disclosure

## Response expectations

- We will acknowledge reports and request clarification as needed.
- Fix timelines depend on severity and complexity.
- Coordinated disclosure is expected.
